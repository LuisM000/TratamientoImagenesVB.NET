Parameters
bmp
Type: System.Drawing Bitmap
Imagen que se quiere guardar. Debe pasarse en formato bitmap.
filtrado (Optional)
Type: System Int32
Tipo de formato que aparecerá como predeterminado a la hora de guardar la imagen. 1 (BMP), 2 (GIF), 3 (JPG/JPEG), 4 (PNG), 5 (TIFF).
Remarks
La imagen guardada se almacena como primera imagen del hacer/deshacer.
