TratamientoImagenes OperAND Method Biblioteca de clases para tratamiento digital de imágenes .NET
Función que realiza la operación AND a los valores de los píxeles de la imagen. Se opera en los valores en los 4 canales ARGB.
Examples
La llamada a la función, asignando la imagen de salida a un Picturebox sería así:

Parameters
bmp
Type: System.Drawing Bitmap
Imagen que se va a transformar. Se debe pasar en formato Bitmap.
Arojo
Type: System Int32
Variable que indica el valor por el cual se operará (con el operador AND) los píxeles del canal rojo.
Averde
Type: System Int32
Variable que indica el valor por el cual se operará (con el operador AND) los píxeles del canal verde.
Aazul
Type: System Int32
Variable que indica el valor por el cual se operará (con el operador AND) los píxeles del canal azul.
AAlfa
Type: System Int32
Variable que indica el valor por el cual se operará (con el operador AND) los píxeles del canal alfa.
omitirAlfa (Optional)
Type: System Boolean
Si la variable es TRUE, no se operará en el canal alfa. En caso de ser FALSE, se hará en AND en función del valor incluido en la variable AAlfa.

Type: Bitmap
Devuelve un bitmap
