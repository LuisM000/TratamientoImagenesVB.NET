TratamientoImagenes filtrosBasicos Method Biblioteca de clases para tratamiento digital de imágenes .NET
Función que calcula la media de las canales RGB para cada píxel, y el valor resultante lo asignan al canal al que se le ha asignado el valor TRUE.
Examples
La llamada a la función, asignando la imagen de salida a un Picturebox sería así:

Parameters
bmp
Type: System.Drawing Bitmap
Imagen que se va a transformar. Se debe pasar en formato Bitmap.
FRojo (Optional)
Type: System Boolean
Si el valor es TRUE, la media calculada se asignará al canal rojo. En caso de ser FALSE el canal rojo será 0.
FVerde (Optional)
Type: System Boolean
Si el valor es TRUE, la media calculada se asignará al canal verde. En caso de ser FALSE el canal verde será 0.
Fazul (Optional)
Type: System Boolean
Si el valor es TRUE, la media calculada se asignará al canal azul. En caso de ser FALSE el canal azul será 0.
Return Value
Type: Object
Devuelve un bitmap
Remarks
Tenga en cuenta que sólo se debería poner un único canal como TRUE.
