TratamientoImagenes
La clase TratamientoImagenes permite crear aplicaciones con multitud de funcionalidades, todas ellas orientadas a tratamiento de imágenes. Con esta clase se pretende englobar todo el proceso de creación de una aplicación de tratamiento de imágenes, desde la propias transformaciones, hasta la adquisición de imágenes o las funciones de deshacer/rehacer.
Examples

Para utilizar la clase en una aplicación, primero se debe hacer referencia con la sentencia Imports:

Ver con coloresCopiar en el PortapapelesImprimir
Imports nombredeaplicacion.Tratamiento
Imports nombredeaplicacion.Tratamiento

A continuación, se puede instanciar a la clase y por ejemplo asignar a un Picturebox una imagen transformada a tonos sepia:

Dim bmp as new bitmap(Picturebox1.image)

ActualizamosImagen
Este delegado, recibe un Bitmap en cuanto se realiza una función dentro de la clase y se genera un evento indicando que se ha modificado la imagen.
Examples

La forma de utilizar el evento (fuera de la clase) es la siguiente: En el load del formulario se pone el siguiente código:

Ver con coloresCopiar en el PortapapelesImprimir
'Asignamos el gestor que controle cuando sale imagen
AddHandler objetoTratamiento.actualizaBMP, New ActualizamosImagen(AddressOf actualizarPicture)
'Asignamos el gestor que controle cuando sale imagen
AddHandler objetoTratamiento.actualizaBMP, New ActualizamosImagen(AddressOf actualizarPicture)

Una vez hecho esto, se crea un procedimiento (por ejemplo), que muestre la imagen en un Picturebox:

ActualizamosNombreImagen
Este delegado, recibe un string en cuanto se realiza una función de abrir una imagen original(desde archivo, FB, recurso web, etc) dentro de la clase y se genera un evento indicando que se ha abierto una imagen original y se dispone de información de ella.
Examples

La forma de utilizar el evento (fuera de la clase) es la siguiente: En el load del formulario se pone el siguiente código:

Ver con coloresCopiar en el PortapapelesImprimir
'Asignamos el gestor que controle cuando se abre una imagen nueva
 AddHandler objetoTratamiento.actualizaNombreImagen, New ActualizamosNombreImagen(AddressOf actualizarNombrePicture)
'Asignamos el gestor que controle cuando se abre una imagen nueva
 AddHandler objetoTratamiento.actualizaNombreImagen, New ActualizamosNombreImagen(AddressOf actualizarNombrePicture)

Una vez hecho esto, se crea un procedimiento (por ejemplo), que muestre la información recibida:

 'Se le pone la información a la cabecera del formulario

End Sub
