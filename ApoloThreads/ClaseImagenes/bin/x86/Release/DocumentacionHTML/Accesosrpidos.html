Accesos rápidos

Abrir: Ctrl + A

Guardar como: Ctrl + G

Deshacer: Ctrl + Z

Rehacer: Ctrl + Y

Zoom +: Ctrl + M // Ctrl + rueda ratón delante

Zoom -: Ctrl + L // Ctrl + rueda ratón atrás

Zoom interactivo: Mayus

Deshacer zoom: Ctrl + 0

Copiar: Ctrl + C

Pegar: Ctrl + V

Histogramas automáticos: Ctrl + H

Ayuda: F1

Ayuda en la web: Ctrl + F1
