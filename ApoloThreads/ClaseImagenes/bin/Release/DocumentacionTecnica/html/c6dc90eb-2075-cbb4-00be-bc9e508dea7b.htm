Función que comparar dos imágenes teniendo en cuenta sus vecinos más próximos y las diferencias entre ellos (los incrementos en sus valores).
Examples

bmp1
Type: System.Drawing Bitmap
Imagen 1 a comparar. Se debe pasar en formato Bitmap.
bmp2
Type: System.Drawing Bitmap
Imagen 2 a comparar. Se debe pasar en formato Bitmap.
DistanciaVecinos (Optional)
Type: System Int32
Variable que indica el número de vecinos que se van a utilizar para la comparación. Para saber el lado del kernel evaluado, se debe calcular (DistanciaVecinos*2+1).
PasoAlto (Optional)
Type: System Boolean
Si esta opción es TRUE, previamente a comparar las imágenes se aplica un filtro de paso alto para acentuar las diferencias en la imagen. En caso de ser FALSE, no se aplica filtro previo.
Grafica (Optional)
Type: System Int32
Esta variable si toma valor 0, no modificará los valores de la comparación. En caso de ser 1, se aplicará a la matriz con los resultados la función math.E^x, y si es 2 se aplicará x^raiz(2).
ComparadorRapido (Optional)
Type: System Boolean
Si esta opción es TRUE, se reducirá el tamaño de la imagen para hacer una comparación rápida pero menos exahustiva. En caso de ser FALSE, se compararán las imágenes con su tamaño original.
Return Value
Type: ArrayList
Devuelve un arraylist con la información de la comparación. En las 4 primeras posiciones devuelve el porcentaje de aciertos del canal rojo, verde, azul y alfa, respectivamente. En 
 las siguientes 4 posiciones, devuelve matrices de dos dimensiones (ancho y alto más pequeño del par de imágenes), con todas las comparaciones píxel a píxel (las diferencias en valor absoluto
 entre los píxeles de la primera imagen con respecto a la segunda). Esta cuatro matrices son del canal rojo, verde, azul y alfa, respectivamente.
Remarks
Si las imágenes son de diferentes tamaños, la imagen de salida tendrá el ancho y alto menor del par de imágenes.
